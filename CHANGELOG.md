# Changelog

## [2.1.0] - 2026-03-30

- Balancing
    - Removed Arc Flash (Zeus) from the starting boons given in Trial of the King (Zeus)
        - Removing this boon should reduce the insane burst damage from the beginning of the trial
    - Added Tranquil Gain (Demeter) and Snow Queen (Demeter) as starting boons given in Trial of Hypothermia (Demeter)
        - Adding 2 boons with Earth elements should now be easier to reach the requirements to activate Rallying Cry (Ares)
- Bug fixes
    - Fixed an issue where Trial of Cinders (Hestia) would not finish after defeating Eris
    - Fixed an issue where Flutter Strike (Aphrodite) was also given alongside Vicious Strike (Ares) as starting boons in Trial of Bloodbath (Ares)
        - Starting boons should now include Flutter Strike (Aphrodite) and Vicious Flourish (Ares) to keep the extra power during the trial, but also keep doors open for Grievous Blow (Ares)

## [2.0.0] - 2026-03-27

- (NEW!) Added 9 new Chaos Trials starring all 10 Wrath boons
- Balancing
    - Hypothermic Shock (Demeter): Added Cold Storage (Demeter) as an eligible requirement
    - Critical Fiasco (Apollo): Reduced delay (0.25 -> 0.10 Sec.) before dodge projectile spawns
    - Eruptive Plating (Hephaestus): Increased blast damage (600 -> 1200)
    - Eruptive Plating (Hephaestus): Increased armor gained (+100 -> +200) when acquiring the boon
    - Heinous Affront (Zeus x Ares): Added Eruptive Plating (Hephaestus) as an eligible requirement
    - Witch Time (Hermes): Increased slow duration (1.5 Sec. -> 2 Sec.)
    - Witch Time (Hermes): Decreased bonus Dodge chance (+15% -> +10%)

## [1.9.0] - 2026-03-06

- Bug fixes
    - Trident Squall (Poseidon): Fixed an issue where a crash could occur by striking a Froth-afflicted foe with some attacks
    - Spurned Patriarch (Zeus): Damage dealt by the lightning bolts now properly display as 'Spurned Patriarch' on the Victory screens
    - Critical Fiasco (Apollo): Damage dealt by the projectile from dodging now properly displays as 'Critical Fiasco' on the Victory screens

## [1.8.0] - 2026-03-03

- Balancing
    - Spurned Patriarch (Zeus): Reduced double strike chance (40% -> 30%)
    - Torrential Submersion (Poseidon): cut from the mod; replaced with...
    - (NEW!) Trident Squall (Poseidon): Your Froth effects are guaranteed, and may fire 3 waves from Ocean Swell.
- Bug fix
    - Hypothermic Shock (Demeter): Fixed an issue where Frostbite could sometimes miss agile foes

## [1.7.1] - 2026-02-25

- Bug fix
    - Wicked Offspring (Hera): Fixed an issue where the bonus Hitch damage from attacking the critter would not be in line with the boon's description

## [1.7.0] - 2026-02-24

- Balancing
    - Lustful Confession (Aphrodite): Reduced internal cooldown (0.25 -> 0.10 Sec.) for the rate at which Omega Casts can be created from Heartthrobs
- Bug fixes
    - Wicked Offspring (Hera): Fixed an issue where Storm Ring (Zeus) could not detect and therefore damage the critter
    - Wicked Offspring (Hera): Fixed an issue where the critter could not automatically be locked-on (i.e. by Torches attacks)

## [1.6.0] - 2026-02-21

- Balancing
    - Critical Fiasco (Apollo): Added a small delay (0.25 Sec.) before dodge projectile spawns, so it doesn't appear instantaneously
    - Lustful Confession (Aphrodite): Added a small internal cooldown (0.25 Sec.) for the rate at which Omega Casts can be created from Heartthrobs
- Bug fixes
    - Wicked Offspring (Hera): Fixed an issue where the critter would be created on Melinoe instead of the cast's location
    - Wicked Offspring (Hera): Fixed an issue where Aspect of Circe (Staff) would sometimes create 1 critter instead of 2
    - Witch Time (Hermes): Fixed an issue where dodging many times would result in an accumulation of wind ambience sounds, which would block other sound cues

## [1.5.0] - 2026-02-18

- (NEW!) Added Hermes' Wrath: "Witch Time"
    - "Whenever you Dodge, make everything else move 75% slower for 1.5 Sec."
    - Added Witch Time (Hermes) as a requirement to complete the "Vengeful Gods" prophecy
- Balancing
    - Torrential Submersion (Poseidon): Reduced bonus wave damage (+100% -> +50%)
    - Hypothermic Shock (Demeter): Reworked effect; Frostbite now deals damage every second an enemy is frozen
        - Frostbite damage increases on every trigger on a frozen enemy
        - Frostbite base damage has been reduced (250 -> 100) to compensate
    - Lustful Confession (Aphrodite): Reduced bonus Heartthrob damage (+50% -> +25%)
